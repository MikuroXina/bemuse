# Bemuse Workshop

The web tool to create Bemuse package files from your own BMS files.

## Features

- Creates a Bemuse package files in `bemuse-data/` in your BMS directory to optimize load speed for Bemuse.
- Creates an external server package file `index.json` to distribute your BMS collection for Bemuse.
- Edits your Bmson data.

## Technical information

This app built on React Router v7 and UI5 Web Components.
